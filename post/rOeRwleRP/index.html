# Excelsior # 精益求精 #
瞎写着玩，别当真

和大佬促膝长谈

2020-06-10

下班之前大佬让我给公司吐槽一下
我去门外抽了两根烟，一时语塞。

平时就我话多，对公司这个不爽那个不满的，到了正儿八经提建议的时候却不知道说什么，可能知道其实并不会有什么改变吧。

大佬和我聊了很多，天花乱坠，应接不暇。
可能我真的要好好思考我个人与公司的关系，和 「我是谁 我在干什么 我要到哪里去」的问题。

说时候，有时候觉得自己想多了。但是根据家族遗传来看，瞎操心是应该会伴随我的终生的梦魇。
既然这样，还不如接受自己的劳心命，多看点书，多干点活呢。

闲着也是闲着 对吧。
